# Coding Agents Are the AI Wedge That Worked

These sources converge on a striking fact: coding turned out to be the killer application that validated the entire agentic AI thesis. Cognition's data shows AI task duration doubling every few months — from seconds to hours — pointing toward autonomous software work measured in days or weeks. Anthropic's revenue exploding from $9B to $30B ARR in four months confirms that model intelligence leaps unlock entirely new categories of demand, not just incremental growth. Karpathy's personal flip — from writing 80% of code himself to writing none — makes this visceral. But the Palo Alto Networks perspective adds a sharp edge: for analytical SaaS companies, this same shift is terminal. The consensus is that coding agents work; the disagreement is whether you're the beneficiary or the casualty.
