# Adversaries Exploit Openness, From Campuses to Code

These sources converge on a shared anxiety: open societies are structurally vulnerable to exploitation by strategic adversaries. China's alleged 20-year plan to co-opt Western universities represents the long-game version — patient, institutional, largely invisible until it worked. The data center moratorium argument makes the same point in real time: domestic political decisions, however well-intentioned, hand adversaries competitive advantage. The deeper problem, as one source frames it, is constitutional — freedoms that define open societies are the precise vectors enemies exploit, raising an uncomfortable tradeoff between liberty and resilience. The AI-or-die framing adds urgency: in military affairs, falling behind isn't a policy failure, it's an existential one.
